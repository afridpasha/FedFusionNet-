# 🏥 NeuroPlex AI - FedFusionNet++

**State-of-the-Art Two-Stage Oral Cancer Detection System**

This is a production-ready AI application for detecting and staging **Oral Squamous Cell Carcinoma (OSCC)** using advanced deep learning and clinical data analysis.

## 🚀 Features

- **Two-Stage Architecture**: CNN image analysis + Clinical data prediction
- **Explainable AI**: Grad-CAM++ and Layer-CAM visualizations
- **Whole Slide Imaging**: Tile-based analysis with heatmaps
- **Survival Analysis**: Kaplan-Meier curves with personalized predictions
- **AI ChatBot**: Multi-language clinical Q&A powered by Gemini/Groq
- **Professional Reports**: Comprehensive PDF clinical reports

## 📖 Documentation

For complete documentation, visit: [GitHub Repository](https://github.com/afridpasha/NeuroPlex-AI)

## 🎯 Quick Start

1. Upload a histopathology image (H&E stained)
2. Fill in patient clinical information (16 features)
3. Click "Analyze" to get comprehensive results
4. Explore XAI visualizations, WSI heatmaps, and survival predictions
5. Chat with AI assistant for clinical insights
